# Simple Python Configuration Management

This Python script provides a basic configuration management system. It loads settings from a JSON file, allows retrieval and modification of these settings, and saves changes back to the file, ensuring consistent application state.

## Language

`python`

## How to Run

1. Save the code as `config_manager.py`.
2. Run from your terminal: `python config_manager.py`.
3. A `config.json` file will be created/updated in the same directory.
